# Archive notice

**This repository is archived.** The Devsper **plugin package registry** (web UI, `devsper reg` CLI flows, and related docs) is **no longer part of the active product**.

- **Plugins** remain supported as normal Python packages: install with `pip` (e.g. from PyPI) and use the `devsper.plugins` entry point. See the main project documentation for [plugins](https://docs.devsper.com/docs/plugins/overview).
- **Devsper Cloud** uses `devsper cloud login`, `devsper cloud run`, and the platform API—not this registry.

This tree is kept for historical reference and operational wind-down only. Do not rely on it for new integrations.
